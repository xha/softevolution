Gerando uma View

Vamos continuar agora com a última camada da nossa aplicação, a view.

php ci first_project

IMPORTANTE: Para gerar a view é necessário que você já tenha gerado a controller.

Logo após executar o comando acima, deverão ser mostradas as camadas que podem ser geradas.

O CrudIgniter exibe as camadas que podem ser geradas, após a escolha da view, são sugeridas as opções baseadas no seu banco de dados.

Quando se está gerando uma view, note que você pode escolher diversos templates. O CRUD é o template mais básico, mas você pode adicionar outros templates ao CrudIgniter. Para isso basta adicionar um novo arquivo .php no diretório templates/views, esse arquivo será um template customizado, mas esse tópico será abordado mais adiante.

IMPORTANTE: Para selecionar diversos templates assim como a validação de campos na controller, basta separar os templates escolhidos com vírgula.

Após a execução, você terá as telas criadas na pasta application/views/users do seu projeto.
