Configurando um projeto

Para configurar um projeto é muito simples, basta executar o CrudIgniter que todas as pastas necessárias serão criadas e os dados para configuração do seu projeto serão perguntados. Não é necessário configurar nenhum XML ou arquivo de configuração!

Primeiro vamos executar o CrudIgniter.

php ci first_project

IMPORTANTE: Para executar o comando acima, é necessário estar na pasta onde se encontra o CrudIgniter.

IMPORTANTE: É necessário ter o php.exe no PATH do seu sistema.

IMPORTANTE: É necessário ter permissão de escrita no diretório projects do CrudIgniter.

Logo após executar o comando acima, deverão ser mostradas as informações de que o projeto foi criado com sucesso.
E já serão solicitados os dados para configurar o banco de dados do projeto.

Após configurar o banco de dados, você já poderá começar a gerar a sua aplicação, mas pressione 4 para sair, pois abordaremos essa parte adiante.

Agora você já deve ter uma pasta dentro do diretório projects, com o nome do seu projeto, e um arquivo de configuração chamado db.php.
