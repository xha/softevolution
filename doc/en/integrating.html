Integrating with CodeIgniter

After all layers created, you just need to integrate with CodeIgniter that your application will be already running.

You must have CodeIgniter installed and configured in a accessible directory by the web server. Then just copy the contents of the respective folder models of CrudIgniter to the folder models of CodeIgniter, then do the same with the controller and view.

The minimum settings that should be made to CodeIgniter are:

  • Configure the base_url in the file application/config/config.php
  • Configure your database in the file application/config/database.php

After performing the above steps, your application is already working!
