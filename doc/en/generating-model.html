Generating a Model

Now that you already have your project created, just rerun the command below to start generating your application!

php ci first_project

Shortly after running the above command, must be shown the layers that can be generated.

**************************************************************
************************* CRUD IGNITER *************************
*********************** IGNITE YOUR CRUD ***********************
**************************************************************
----------------------------------------------------------------

What do you want to generate?
1. MODEL
2. VIEW
3. CONTROLLER
4. EXIT
[1] > 1

What Model do you want to generate?
1. users
> 1
Model first_project\application\models\users_model.php was successfully created!
CrudIgniter exit.

The CrudIgniter displays the layers that can be generated, after the choice of which layer generate, your database will be read and will be showed the options found. In our case we only have the option of users that corresponds to a table in the database.

Upon execution, you will have a new file created in the folder application/models of your project.
