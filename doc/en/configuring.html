Configuring a project

To configure a project is very simple, just run the CrudIgniter that all the necessary folders will be created and data for configuring your project will be asked. There is no need to configuration file or XML!

First we run the CrudIgniter.

php ci first_project

IMPORTANT: To run the above command, you must be in the folder where the CrudIgniter is.

IMPORTANT: You must have php.exe in the PATH of your system.

IMPORTANT: You must have write permission on the directory of projects on CrudIgniter.

Shortly after running the above command, the information must be shown that the project was successfully created. And the data to configure the project's database will be required.

After configuring the database, you can already begin to generate your application, but press 4 to leave, because we discuss this part further.

You should now have a folder inside the directory projects, with the name of your project, and a configuration file called db.php.
