Custom validation

When you're generating a controller you have several rules to validate the fields. But you can always use multiple rules together, always having to type for example: 3,5,6. Where in this case you want to add the following rule.

required|min_length[5]|max_length[10]

With CrudIgniter you can create a custom type of validation! To do this simply open the file crudigniter/validation/ValidationTypes.php

IMPORTANT: Note that this file have all the validation rules supported by CodeIgniter.

Now let's create a custom validation rule to avoid have to type all the time 3,5,6.

Adds another position in the array ValidationTypes with your customized rule.

array(
        'Name' => 'Custom validation',
        'Rule' => 'required|min_length[%s]|max_length[%s]'
)

IMPORTANT: Note that when a rule takes parameters, you should add %s in the place where the parameter should be placed. Because in that way CrudIgniter will require the parameter to that place.
