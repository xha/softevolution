Generating a View

Let's continue now with the last layer of our application, the view.

php ci first_project

IMPORTANT: To generate the view, you must have already create the respective controller.

Shortly after running the above command, must be shown the layers that can be generated.

CrudIgniter exit.

The CrudIgniter displays the layers that can be generated, after the choice of view, the options are suggested based on your database.

When you are generating a view, note that you can choose several templates. The CRUD is the most basic template, but you can add other templates to CrudIgniter. To do this simply add a new file .php in the directory templates/views, this file will be a custom template, but this topic will be discussed later.

IMPORTANT: To select multiple templates as well as validation of fields in the controller, simply separate the templates chosen with commas.

Upon execution, you will have the screens set up in the folder application/views/users of your project.
